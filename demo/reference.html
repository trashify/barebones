Reference

The reference doc tells you about the components included on the latest version of Barebones, and how to use them. For checking how to download and install Barebones, read the getting started guide.

Typography

Barebones uses Roboto as the default font with Raleway used as the font for headers. The default font-size is 16px, and the default font-weight is 300 (light). Barebones comes with bold tag, italic, underline, links and small text. The default text color is #373d3f.

Buttons

Add a .button class to transform your div, a or button element into a normal button, or add the .btn--outline class to transform your element to an outline button.

Outline button
No Outline button
<div class="button">Default button</div>
<div class="button--outline">Outline button</div>
<div class="button--no-outline">No Outline button</div>

Grid

Barebones comes with flexboxgrid to provide a responsive and modern grid system. Please see flexboxgrid's documentation for how to use the grid.
